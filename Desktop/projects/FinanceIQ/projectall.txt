🎯 What This Project Does
Problem Solved
People struggle to analyze bank statements manually - it's time-consuming, error-prone, and doesn't provide actionable insights.

Solution
FinanceIQ uses a 6-node LangGraph AI agent to automatically:

Extract transactions from PDF/CSV
Categorize each transaction (12 categories)
Analyze income, expenses, trends
Generate insights about spending patterns
Provide advice with specific $ savings estimates
Self-reflect to ensure quality recommendations
Result: 30-second analysis with personalized dashboard, charts, and actionable recommendations.

Setup Steps:

Sign up (no credit card)
Create API Key
Copy key (starts with gsk_)
2. Backend Setup (3 min)
powershell
cd backend
pip install -r requirements.txt
copy .env.example .env
notepad .env  # Add: GROQ_API_KEY=gsk_your_key_here
3. Frontend Setup (2 min)
powershell
cd frontend
npm install
4. Start Servers (30 sec each)
powershell
# Terminal 1 - Backend
cd backend
uvicorn main:app --reload --port 8000
 
# Terminal 2 - Frontend
cd frontend
npm run dev

📈 How It Works (Technical)
Architecture
Upload → FastAPI Backend → LangGraph Agent (6 Nodes) → React Dashboard
              ↓
         Groq API (Free)
       LLaMA 3.3-70B
6-Node Pipeline
Extraction (5-10s): Parse PDF/CSV → structured transactions
Categorization (5-10s): AI classifies into 12 categories
Analysis (1-2s): Pandas calculates metrics
Insights (3-5s): AI identifies patterns
Advice (5-8s): AI generates recommendations
Reflection (3-5s): AI quality-checks advice
Total: 20-40 seconds per analysis

Tech Stack
Backend: FastAPI + LangGraph + Groq API
Frontend: React + Vite + Tailwind + Recharts
AI: LLaMA 3.3-70B (via Groq free tier)
Analysis: pandas + pdfplumber
